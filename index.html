¿Porqué esta web, si todo está en el blog?

Este github page esta destinada a recoger las entradas más interesante de mi blog y ordenarlos por temática, haciendo mas accesible al contenido que vaya creando, dándome la oportunidad de mejorarlo con el tiempo, sin que se pierda con el tiempo. Destinada a compartir de una forma práctica el conocimiento que yo he adquirido con el tiempo o bien, mediante práctica, cómo por lecturas.

Considero muy importante las bases, algoritmia el clean code y el refactoring, por eso veo una buena oportunidad de recopilar y practicar algunos aspectos que en el día a día no puedo. Podría considerar este pequeño rincón como mi laboratorio personal.

Tener paciencia, seguramente, este en continua evolución y generación de contenido, aunque siempre podéis preguntar o animarme hacer algún patrón, refactoring o consejo, mandando un email a mi dirección de correo electrónico.

Espero que os guste y practiquéis mucho, porque el secreto esta en eso, la practica.
